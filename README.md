## About

A clean, dark-themed developer portfolio built with vanilla HTML, CSS, and JavaScript. Features smooth animations, a typing effect, and a fully responsive layout.

## Features

- Typing animation in hero section
- Scroll-triggered fade-in animations
- Responsive mobile layout with hamburger menu
- Project cards with live + GitHub links
- Contact form
- Dark theme with customizable accent colors

## Run Locally

```bash
git clone https://github.com/chaudharybhakti0-bot/portfolio
cd portfolio
# Open index.html in your browser
```

No dependencies. No build step. Just open `index.html`. ✅

---

## Customization

To change the color theme, edit the CSS variables in `style.css`:

```css
:root {
  --accent: #FF3B5C;   /* primary accent color */
  --accent2: #FF00AA;  /* secondary accent color */
  --bg: #110A0A;       /* background */
}
```
